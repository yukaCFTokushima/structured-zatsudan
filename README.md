# structured-zatsudan
このリポジトリは、地域や実践現場で行われている“雑談”のプロセスを構造化・再利用可能なかたちで記録することを目的としています。話題の遷移、問いかけの類型、参加者の関係性、感情の動きなどをテキストデータやCSV形式で整理し、雑談の「空気」や「場づくり」の技術を可視化しました。瀬戸先生の授業で紹介された“オープンな技術と公共性”の話を受け、雑談もまたローカルな営みであると同時に、形式知に変換し得る「参加型の社会技術」だと捉えています。この記録が、他地域や他分野での雑談実践のヒントとなり、「誰もが関われる公共」の設計に役立てばと考えています。今後は、話題分類やメタデータ付与によって検索性を高め、雑談のログを地域の文化資産として活用できる仕組みを模索していきます。
